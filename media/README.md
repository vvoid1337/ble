# media/

Статические медиа. Эта папка коммитится в репозиторий и раздаётся через `raw.githubusercontent.com` (см. `MEDIA_BASE_URL`
в `app/config.py`). В БД хранятся **относительные** пути внутри этой папки.

> ⚠️ Сами файлы ассетов пока НЕ добавлены — ниже описана ожидаемая структура и
> имена. До загрузки реальных фото/видео ссылки на медиа будут отдавать `404`.

## Структура

```
media/
  znamensky-sobor/
    cover.jpg
    01.jpg
    02.jpg
  krasnaya-ploshchad/
    cover.jpg
    01.jpg
    02.jpg
  kurskaya-krepost/
    cover.jpg
    01.jpg
    02.jpg
```

Имена файлов должны совпадать с путями в `content/*.json` (поля `cover_image`
и `gallery[].src`). Видео кладите рядом и указывайте `"type": "video"` в галерее.

## Переезд на CDN

Поменяйте переменную окружения `MEDIA_BASE_URL` — пути в БД останутся прежними.
